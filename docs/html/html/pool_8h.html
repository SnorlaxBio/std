Detailed Description

Header file that declares structures and method related to the buffer pool.

This header file declares a function table structure that defines the main structure and main methods of the buffer pool. Additionally, the gen(size: uint64, page: uint64): pool method that dynamically creates a basic buffer pool object and a macro that calls member functions are defined.
