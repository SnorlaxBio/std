Detailed Description

buffer pool structure

The buffer pool structure is an inherited structure with object_t as its parent structure. It is a basic buffer pool that is created by defining the page size and number of buffers for dynamic memory allocation at the time of creation. Library users can inherit this structure to define and use new structures.
